# directus-index-with-meilisearch

Directus custom extension which provice Operation for flow which allow to index collection in meilisearch

## Install

```
npm run build
```

Deploy package.json and dist folder in

```
directus/extensions/directus-index-with-meilisearch/
```

## Configuration

- Add MEILISEARCH_URL in directus environnement variables
- Add EXTENSIONS_AUTO_RELOAD=true in directus environnement variables
